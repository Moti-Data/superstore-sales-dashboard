# 🏪 Superstore Sales Dashboard

Excel-based sales analytics dashboard built on 9,994 real orders.

## 📊 What's Inside
- Data Cleaning & Transformation
- KPI Summary (Revenue, Profit, Orders, Customers)
- Product & Sub-Category Analysis
- Monthly & Yearly Trend Charts
- 6 Business Recommendations

## 🛠️ Tools Used
Excel | Python (pandas, openpyxl)

## 📁 Dataset
Sample Superstore Dataset | 2014–2017 | USA
